### Basic File Operations



Open a file: **vi filename**



Save changes: :**w**



Save and exit: :**wq or :x**



Exit without saving: :**q!**



Insert Mode



Enter insert mode at cursor: **i**



Enter insert mode after cursor: **a**



Insert at the beginning of the line: **I**



Insert at the end of the line: **A**



Insert a new line below: **o**



Insert a new line above: **O**



Exit insert mode: **Press ESC**



Navigation



Move left: **h**



Move down: **j**



Move up: **k**



Move right: **l**



Move to the beginning of the line: **0**



Move to the end of the line: **$**



Move forward one word: **w**



Move backward one word: **b**



Go to the first line: **gg**



Go to the last line: **G**



Go to line number n: **nG**



Editing



Delete a character: **x**



Delete the current line: **dd**



Delete n lines: **ndd**



Delete from cursor to the end of the line: **D**



Undo last change: **u**



Redo last undone change: **CTRL + r**



Replace a character: **r**



Change a word: **cw**



Change to the end of the line: **C**



Copy and Paste



Copy (yank) a line: **yy**



Copy n lines: **nyy**



Paste after the cursor: **p**



Paste before the cursor: **P**



Search and Replace



Search for a pattern: **/pattern**



Find next occurrence: **n**



Find previous occurrence: **N**



Replace all occurrences of a pattern: **:%s/old/new/g**



Display Options



Show line numbers: **:set number**



Hide line numbers: **:set nonumber**



Miscellaneous



Scroll down half a screen: **CTRL + d**



Scroll up half a screen: **CTRL + u**



Redraw the screen: **CTRL + l**
